# 🪄 Neramit-AI (เนรมิต AI)

Neramit-AI คือแพลตฟอร์มที่ช่วยให้นักเขียนและครีเอเตอร์สามารถ "เนรมิต" นิยายและโครงเรื่องได้อย่างรวดเร็วผ่าน AI เพียงแค่ป้อนไอเดียหรือคีย์เวิร์ดสั้นๆ ระบบจะประมวลผลและเขียนเนื้อหาออกมาในโทนภาษาที่สละสลวย พร้อมนำไปใช้งานได้ทันที 

## ✨ Core Features
* **One-Prompt Generation:** พิมพ์ไอเดียตั้งต้น (เช่น GL, การเมือง, อายุต่างกัน) AI จะสร้างเนื้อหาให้ครบถ้วน
* **Tone & Style Customization:** หากผลลัพธ์ยังไม่ถูกใจ สามารถกดเลือกปรับเปลี่ยนโทนภาษา (เช่น ดราม่า, โรแมนติก, จริงจัง) ได้แบบ Real-time
* **Ready-to-Use Output:** โครงสร้างประโยคและภาษาถูกปรับแต่งมาให้เหมือนมนุษย์เขียน พร้อมให้ Copy ไปใช้งานหรือเกลาต่อได้ทันที
* **Project Management:** ระบบจัดการหลังบ้านสำหรับบันทึกโครงเรื่อง ตัวละคร และเนื้อหาแต่ละตอน (Chapter by Chapter)
